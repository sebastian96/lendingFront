### Clone repository
``` 
git clone https://github.com/sebastian96/lendingFront.git
```

### Frontend Tools:
- React
- Axios
- Node JS / npm

### Run Frontend
``` 
cd lendingFront/client
npm install
npm run start
```

### Backend Tools:
- Python 3.9.7
- Flask
- Flask-cors

### Run Backend
``` 
cd lendingFront/loan-service
pip install -r requirements.txt
python app.py
```
